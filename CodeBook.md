Once readed into variables, the data was transformed following these process:

* Content from 'X_train.txt' is appended to content from 'X_test.txt' and stored into variable 'both_sets'
* Content from 'y_train.txt' is appended to content from 'y_test.txt' and stored into variable 'activities'
* Content from 'subject_train.txt' is appended to content from 'subject_test.txt' and stored into variable 'subjects'
* Since 'subjects' is a data.frame of one column, we assign a column name 'Subject'
* Read the content of 'features.txt' file into a data frame with 2 columns (the name of the variable is 'features'): a numerical id, and a label (string) of the columns of the main dataset.
* Set the column names of the main dataset (in variable 'both_sets') using the second column of 'features' variable.
* Create a subset of 'both_sets' named 'tidy_set'. It only contains columns related to measurements on the mean and standard deviation
* Create a factor object using the 'activities' data frame
* Create a data frame called 'tidy_set_with_activity' that contains all data from 'tidy_set', 'Activity' and 'subjects'
* Create an independent tidy data set with the average of each variable for each activity and each subject. This data set is stored in variable 'second_data_set'
* Finally, store the content of 'second_data_set' in a file called 'tidy_dataset.txt'
